
# 🎭 Affectra – AI-Powered Emotion-Based Content Recommendation System
live link https://affectra-koea.onrender.com

## Overview

Affectra is an AI-powered full-stack web application that analyzes facial expressions in real time and delivers personalized entertainment recommendations based on the user's emotional state.

Using computer vision and facial landmark analysis, the system detects emotions such as Happy, Sad, Angry, Surprised, Sleepy, and Pout. Based on the detected emotion, Affectra recommends relevant songs and memes while maintaining an emotion history dashboard for behavioral insights.

## Problem Statement

Most recommendation systems rely on user clicks, searches, or historical preferences.

Affectra introduces a different approach by leveraging real-time facial emotion recognition to create an adaptive and emotionally aware recommendation experience.
